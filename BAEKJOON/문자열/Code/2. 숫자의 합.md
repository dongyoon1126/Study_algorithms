# 2. 숫자의 합
## 알고리즘 분류
* 수학
*  구현
*  문자열

## 문제
* N개의 숫자가 공백 없이 쓰여있다. 이 숫자를 모두 합해서 출력하는 프로그램을 작성하시오.

## 입력
* 첫째 줄에 숫자의 개수 N (1 ≤ N ≤ 100)이 주어진다. 둘째 줄에 숫자 N개가 공백없이 주어진다.

## 출력
* 입력으로 주어진 숫자 N개의 합을 출력한다.

## 예제
|입력|출력|
|:---|:---:|
|1<br>1|1|
|5<br>54321|15|
|25<br>7000000000000000000000000|7|
|11<br>10987654321|46|
 
## Code
```swift
let input1 = Int(readLine()!)!
let input2 = readLine()!

var count = 0

for i in input2 {
    count += Int(String(i))!
}

print(count)
```
