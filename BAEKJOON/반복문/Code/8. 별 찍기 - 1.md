# 8. 별 찍기 - 1
## 알고리즘 분류
* 구현

## 문제
* 첫째 줄에는 별 1개, 둘째 줄에는 별 2개, N번째 줄에는 별 N개를 찍는 문제

## 입력
* 첫째 줄에 N(1 ≤ N ≤ 100)이 주어진다.

## 출력
* 첫째 줄부터 N번째 줄까지 차례대로 별을 출력한다.

## 예제
|입력|출력|
|:---:|:---|
|5|* <br> ** <br> *** <br> **** <br> *****|

## Code
```swift
let input = Int(readLine()!)!

for i in 1...input {
    print(String(repeating: "*", count: i))
}
```
